## ¿Qué es?
Vocaloid Music Browser es una página web para buscar información sobre canciones, álbumes y artistas del mundo [VOCALOID](https://es.wikipedia.org/wiki/Vocaloid). Todos los datos son obtenidos desde la [API REST](https://vocadb.net/swagger/index.html) de [VocaDB](https://vocadb.net/).

## Tecnologías usadas
- Laravel 12
- jQuery 4.0.0
- Bootstrap 5.3.8
- SASS 1.94.2
- Vite 7.0.7
- jqueryUI 1.14.1
